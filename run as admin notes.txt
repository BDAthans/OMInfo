Set program to run as Administrator for all users on program:
C:\Users\BriaAt\Desktop\testforadmin.exe


Looked in regedit for registry keys and values and found:
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\AppCompatFlags\Layers

REG_SZ type value shows:
Name = C:\Users\BriaAt\Desktop\testforadmin.exe
Data = RUNASADMIN


Cmd Line for adding registry key works:
reg add "HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\AppCompatFlags\Layers" /v C:\Officemate\Omate.exe /t REG_SZ /d "~ RUNASADMIN"



Windows Registry Export for setting program to run as Administrator to .reg:



Windows Registry Editor Version 5.00

[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\AppCompatFlags\Layers]
"C:\\Users\\BriaAt\\Desktop\\testforadmin.exe"="RUNASADMIN"
